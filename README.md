# AI内容检测技能

两个用于检测AI幻觉和虚假内容的技能，支持OpenClaw框架。

## 技能列表

### 1. AI内容幻觉检测器 (ai-hallucination-detector)

专业的AI幻觉内容检测工具，支持：
- 伪科学表述检测
- 科学词汇滥用识别
- 专业领域风险分析（金融、法律、医疗、教育）
- 数据逻辑冲突检测
- 自动词库学习

**版本**: v1.4.0

**使用方法**:
```bash
cd ai-hallucination-detector
python scripts/hallucination_detector.py
```

### 2. AI内容说谎检测器 (ai-liar-detector)

趣味化包装的说谎内容检测工具，支持5种侦探角色：
- 🔍 狄仁杰（唐朝神探）
- ⚖️ 包青天（北宋名臣）
- 🕵️ 柯南（日本侦探）
- 🧠 福尔摩斯（英国侦探）
- 🎬 秦风（唐人街探案）

**版本**: v1.0.0

**使用方法**:
```bash
cd ai-liar-detector
python scripts/liar_detector.py
```
