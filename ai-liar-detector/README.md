# AI内容说谎检测器

以多种侦探风格，检测AI生成内容中的谎言、伪科学和虚假宣传。

## 功能特点

- 🎭 **五种风格**：狄仁杰、包青天、柯南、福尔摩斯、秦风
- 🔬 **伪科学识别**：识别量子养生、DNA修复等伪科学营销
- 📊 **数据核验**：核实AI生成数据的合理性
- ⚠️ **风险分级**：区分欺诈性内容、虚假成分和可信内容
- 🎮 **趣味性强**：独特的角色扮演体验

## 支持角色

| 角色 | 时代 | 特点 |
|------|------|------|
| 狄仁杰 | 唐朝 | 神探风格，"元芳，此事必有蹊跷..." |
| 包青天 | 宋朝 | 名臣风格，"王朝马汉，开堂审案！" |
| 柯南 | 现代日本 | 侦探风格，"真相只有一个！" |
| 福尔摩斯 | 维多利亚英国 | 绅士风格，"Elementary, my dear Watson." |
| 秦风 | 现代 | 天才侦探，"这案子有点意思..." |

```

```

### 秦风/唐人风格
```
/秦风 检测：石墨烯养生床垫，睡出健康好身体
```

### 柯南风格
```
/柯南 检测：7天彻底治愈近视，永不复发！
```
